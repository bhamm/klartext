Klartext - Leichte Sprache

Leer lassen für Standard-Endpoint

Einstellungen

Große Schrift

Anleitung

So verwenden Sie Klartext:

  • Text markieren, den Sie übersetzen möchten
  • Rechtsklick auf den markierten Text
  • Wählen Sie "In Leichte Sprache übersetzen"

Die Übersetzung erscheint in einem Overlay-Fenster.
