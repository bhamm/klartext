Anleitung

So verwenden Sie Klartext:

Text übersetzen

  • Text markieren, den Sie übersetzen möchten
  • Rechtsklick auf den markierten Text
  • Wählen Sie "Markierten Text in Leichte Sprache übersetzen"

Artikel übersetzen

  • Rechtsklick auf eine beliebige Stelle der Webseite
  • Wählen Sie "Artikel in Leichte Sprache übersetzen"
  • Bewegen Sie die Maus über den gewünschten Textbereich
  • Klicken Sie auf den hervorgehobenen Bereich

Ganze Seite übersetzen

  • Rechtsklick auf eine beliebige Stelle der Webseite
  • Wählen Sie "Ganze Seite in Leichte Sprache übersetzen"
  • Die Übersetzung erfolgt abschnittsweise mit Fortschrittsanzeige
  • Optional: Original und Übersetzung nebeneinander anzeigen

Vorlesefunktion

  • Klicken Sie in der Übersetzung auf "Vorlesen"
  • Der Text wird vorgelesen und das aktuelle Wort hervorgehoben
  • Klicken Sie erneut zum Pausieren/Fortsetzen
